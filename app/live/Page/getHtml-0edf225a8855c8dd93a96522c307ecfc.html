Título da seção

Primeiro parágrafo começa aqui.

Usa o editor ao lado para editar HTML ("getHtml" - estrutura e conteúdo), Javascript ("onLoad" - interação com usuário) e CSS ("getCss" - visual).

Desculpa, isto é um protótipo. Se as letras não estiverem aparecendo, alterna entre as do editor.

Edição ao vivo/assistida por agentes

Experimenta também a edição assistida por agentes, direcionando eventos do DOM para um agente, por exemplo:

<button onclick="getAgent(this).see('novaAcao', 'args');">Clique para definir uma ação</button>
