NaiveMind

  • Agente reativo.
  • Capaz de aprender.
  • Ingênuo, sem senso crítico.
  • Strategy pattern.

Teste de inteligência

O teste abaixo verifica se o agente é inteligente, ou seja, é capaz de adquirir e aplicar conhecimento.

(async () => { let agente = new NaiveMind('Joe').body; let resposta = document.querySelector( '#resposta_teste_inteligencia'); resposta.innerText += '\n' + agente; // Teste falha, o agente não sabe de nada. assert.equal(await agente.see('soma', [1, 2]), 3); // Ensinando uma nova ação await agente.see('set', ['soma', new class soma { act(args, resolve, reject) { resposta.innerText += '\n\n' + this.agent; let soma = 0; for (let n of args) { soma += n; } resolve(soma); } }()]); // Teste bem sucedido, o agente aprendeu! assert.equal(await agente.see('soma', [1, 2]), 3); })()

Problema - Desenvolvimento inteligente

Soluções baseadas em inteligência artificial também sofrem com o ciclo "editar-salvar-atualizar".

  • Se o agente falhar, é necessário reiniciar o programa, matar o processo.

Solução

Agentes precisam poder errar sem que matem seu processo por isso.

  • Ver e alterar código-fonte de programas em execução.
  • Desenvolvimento de software como treinamento de agentes em execução.

Ceed - Clever Editing

  • Treinamento em tempo de execução.
  • Ler/Escrever.
  • Interpretar.

Vantagens competitivas - Ceed

Clever Editing - Abordagem top-down assistida por agentes:

(async () => { let agente = await Ceed('Pizzaiolo'); await agente.see('set', ['pizza', new (function () { this.act = async function (args, resolve, reject) { let agent = this.agent; let pizza = '\nPizza do ' + agent; pizza += '\n' + await agent.see('massa', args); if (await agent.see('perguntarQueijo', args)) { pizza += '\n' + await agent.see('queijo', args); } pizza += '\n' + await agent.see('recheio', args); resolve(pizza); }; })()]); let resposta = document.querySelector('#resposta_ceed'); resposta.innerText += await agente.see('pizza', 'atum'); })();
